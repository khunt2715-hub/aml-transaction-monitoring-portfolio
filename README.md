# AML Transaction Monitoring Portfolio

## Overview
Rule-based AML transaction monitoring system with alert investigations and risk analysis

## AML Typologies Covered
- Rapid Movement of Funds
- High-Risk Jurisdiction Exposure
- Dormant Account Reactivation

## Data
Synthetic customer and transaction data was generated to simulate realistic banking behaviour

## Methodology
- Created rule-based alert logic
- Generated alerts from transactional data
- Conducted manual investigations
- Evaluated alert effectiveness and false positives

## Tools Used
-  SQL / Excel
- GitHub
  
## Key Outcomes
- Alert generation across four AML typologies
- Investigation narratives with escalation decisions
- Alert tuning insights to reduce false positives
  
## Limitations & Future Enhancements
- Synthetic data constraints
- No real-time monitoring
- Potential ML-based alert prioritization
